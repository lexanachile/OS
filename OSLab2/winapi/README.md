# WinAPI Threads Project

## Описание
Проект демонстрирует работу с потоками WinAPI для параллельного вычисления:
- Минимального и максимального элементов массива
- Среднего арифметического элементов массива
- Замены min/max элементов на среднее значение

## Технологии
- C++98
- WinAPI
- CMake
- Boost.Test (для тестирования)

## Сборка
```bash
mkdir build && cd build
cmake ..
cmake --build .
